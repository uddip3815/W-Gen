# W-Gen
# Installation and Usage[Linux] : 
* `apt-get update -y`
* `apt-get upgrade -y`
* `apt-get install python -y`
* `apt-get install git -y`
* `git clone https://github.com/uddip3815/W-Gen.git`
* `cd $HOME`
* `ls`
* `cd W-Gen`
* `ls`
* `python lazybee.py`
# Installation and Usage[Termux] : 
* `apt-get update -y`
* `apt-get upgrade -y`
* `apt-get install git -y`
* `cd $HOME`
* `git clone https://github.com/uddip3815/W-Gen.git`
* `ls`
* `cd W-Gen`
* `chmod +x *`
* `./install`
* `./wgen`
